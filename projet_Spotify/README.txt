======================== React App Spotifly =========================

Pour lancer l'application:

1) Se rendre dans le dossier front-end depuis le terminal
2) Aller dans le dossier spotifly toujours depuis le terminal, puis src , et ecrire:
npm run start 
3) Visualisez l'interface de l'application sur le navigateur afin d'intéragir avec.

Après avoir accéder aux détails d'une playlist, veuillez rafraichir la page en faisant ctrl+R afin de pouvoir effectuer une nouvelle recherche dans l'application.
